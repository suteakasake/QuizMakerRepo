【 分岐条件 】
-- 起動処理 --
新規作成の場合は、問題形式ごとの初期値を構造体に格納する

-- テーブルロード --
問題形式ごとのchoiceValueの値分セルを作成する
TextFieldにTag（cellの番号）を格納する
問題形式ごとの構造体内のchoiceValueから値を取得してセルに表示する


-- 新規作成 --
・cellCountの個数分cellを作成する
    ・選択肢の数のセルが押される
        → 構造体に「問題文」「選択肢数」「答えの数」「選択肢の文字を格納する」
    ・選択肢が決定する
        → 「選択肢数」「セルの個数」を更新する
        → 構造体の「選択肢数」を変更し、「選択肢の文字」に増加分の空値を格納する、減少した場合は構造体から減少分削除する

    ・問題形式の変更を押下
        → 構造体に「問題文」「選択肢数」「答えの数」「選択肢の文字を格納する」
        → quizTypeを変更する
    
    ・TextViewの入力処理
        → 入力完了後、TextViewの値を、構造体のquestionに格納する

    ・TextFieldの入力処理
        → 入力完了後、TextFieldのTagを元に、構造体のchoiceValue配列に入力値を格納する
    
・編集



・送信処理

initStructSetting()     // 問題形式ごとの初期値を構造体に格納する処理
numberSettingBefore()   // 選択肢の数のセルが押された際の処理
    入力値を構造体に格納する処理
numberSettingAfter()    // 選択肢の数が変更された際の処理
    構造体に値を増加、削除する処理
editTextField()         // TextFieldが編集完了した際の処理
